Food Data Analysis Dashboard (Power BI)

Project Overview
This project focuses on analyzing food delivery data extracted from an XML file and transforming it into meaningful insights using Power BI.

The dashboard provides a clear understanding of business performance including revenue trends, order patterns, and delivery efficiency.

Key Features
- 📈 Total Revenue Analysis
- 📦 Total Orders Tracking
- 💰 Average Order Value (AOV)
- ⏱️ Average Delivery Time
- 📅 Date-wise Analysis (Day, Month, Year Hierarchy)
- 📍 Customer & Order Behavior Insights

Tools & Technologies Used
- Power BI (Data Visualization)
- XML Data Source
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)

Data Processing Steps
1. Imported XML data into Power BI
2. Cleaned and transformed data using Power Query
3. Created relationships between tables
4. Built DAX measures for KPIs
5. Designed interactive dashboard with filters and slicers

Insights Generated
- Identified peak order periods
- Analyzed revenue trends over time
- Evaluated delivery performance
- Found patterns in customer ordering behavior

Conclusion
This dashboard helps in making data-driven decisions for improving business performance and customer satisfaction.
